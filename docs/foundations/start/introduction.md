# 前端基础 · 介绍

> 从前端最基础的内容出发，理解前端的本质

## 这一章讲什么

前端开发的本质，是用 **HTML、CSS、JavaScript** 三种语言描述一个可交互的页面。在各种框架和工具盛行的今天，理解这三者的底层运行机制，依然是区分"会用"与"真正懂"的分水岭。

这一章不追求面面俱到，只聚焦那些真正影响你写代码、排查问题、理解框架的核心概念。

## 内容规划

| 模块                | 内容概述                                             |
| ------------------- | ---------------------------------------------------- |
| HTML 语义化         | 结构与语义的关系，SEO 与可访问性的底层逻辑           |
| CSS 盒模型与布局    | 从 BFC 到 Flexbox / Grid，彻底搞清楚"为什么它跑偏了" |
| JavaScript 运行机制 | 事件循环、调用栈、微任务与宏任务                     |

## 学习建议

- 遇到不理解的行为，优先去浏览器 DevTools 里**亲手验证**，而不是背结论
- 框架只是对原生能力的封装，理解原生才能真正读懂框架源码
- 本章内容适合配合实际项目中遇到的问题反复回看
